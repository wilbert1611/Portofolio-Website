Image Denoising with Autoencoders

I implemented an Autoencoder model to reduce noise in pistachio seed images. The model consists of an encoder, which compresses the image into a lower-dimensional representation, and a decoder, which reconstructs a cleaner version of the image. After training, the Autoencoder successfully reduced noise while preserving key visual features, demonstrating its potential for image processing and restoration tasks.
